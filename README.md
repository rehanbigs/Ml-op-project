# Ml-op-project
Ml ops project of digital text recognizer and building on top of it to make it work on forms.

## QuickStart

### Environment Setup
Create a python virtualenv and run the following commands:

```
python --version    # Python 3.10.14
sudo apt-get install python3-tk
pip install pillow opencv-python torch numpy matplotlib smart_open tk
```
open the notebook `lab07/notebooks/main.ipynb` and Execute all the cells
